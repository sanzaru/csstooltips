About

CSS Tooltips is a simple CSS / SASS library to create CSS3 tooltips without any Javascript included. Click here for a Demo

Build the library 

$ git clone https://github.com/sanzaru/csstooltips.git
$ cd csstooltips
$ npm install
$ grunt

Options

cssttp (required):
The class to setup the tooltip.
fade-in (optional):
If set the tooltip will fade in.
right (optional):
If set the tooltip will be shown on the right side.
left (optional):
If set the tooltip will be shown on the left side.
top (optional):
If set the tooltip will be shown above.
